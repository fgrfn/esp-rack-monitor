# Changelog

## 2026-06-02 – Initial Version

- Initiales ESPHome-Projekt für LilyGO T-Display-S3 angelegt.
- Projektname auf **ESP Rack Monitor** gesetzt.
- Konfiguration unter `configs/esp-rack-monitor.yaml` abgelegt.
- Layout enthält:
  - Rack-Übersicht
  - Stromverbrauch
  - Temperaturen
  - CPU/RAM/DISK
- Auto-Rotation und Button-Hinweise integriert.
- Farbige Statuslogik für Strom, Temperatur, CPU, RAM und Speicherplatz integriert.
